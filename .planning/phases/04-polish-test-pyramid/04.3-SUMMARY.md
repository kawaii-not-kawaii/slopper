## Issues Encountered

**Coil 3.0.4 / AGP 8.7.3 lint incompatibility (pre-existing):** The Coil 3.0.4 `lint.jar` triggers a `NegativeArraySizeException` in AGP 8.7.3's `LintJarApiMigration.Frame.merge` during lint jar loading. This affects all modules that transitively depend on Coil. The crash was introduced when DEPS-05 bumped Kotlin to 2.2.20 (May 16) but was not detected at the time because the same commit added `lint { disable.addAll(...) }` for other unrelated crashes. The `checkAdditionalJars` property does not exist in AGP 8.7's Lint DSL, so there is no simple in-build suppression. Resolution: upgrade Coil to a version that ships an AGP 8.7-compatible lint jar (likely 3.1.x or later).

**Configuration cache stale state:** After manually cleaning individual modules, Gradle's configuration cache held stale task graphs. Resolved by adding `--no-configuration-cache` to all subsequent build commands.
