    7. **Define `HighlightedText`** private composable (simple `AnnotatedString` approach):

    ```kotlin
    @Composable
    private fun HighlightedText(
        text: String,
        highlight: String,
        normalColor: Color,
        highlightColor: Color,
        highlightBg: Color,
        style: TextStyle,
    ) {
        if (highlight.isBlank()) {
            Text(text, style = style, color = normalColor)
            return
        }
        val idx = text.indexOf(highlight, ignoreCase = true)
        if (idx < 0) {
            Text(text, style = style, color = normalColor)
            return
        }
        val annotated = buildAnnotatedString {
            withStyle(SpanStyle(color = normalColor)) { append(text.substring(0, idx)) }
            withStyle(SpanStyle(color = highlightColor, background = highlightBg,
                                fontWeight = FontWeight.SemiBold)) {
                append(text.substring(idx, idx + highlight.length))
            }
            withStyle(SpanStyle(color = normalColor)) {
                append(text.substring(idx + highlight.length))
            }
        }
        Text(annotated, style = style)
    }
    ```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User search input → filter function | Free-text input; processed client-side against static list only |
| Search result route → navigation | Route string from static list; can only match known Routes.* constants |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-05 | Tampering | Static SettingsSearchIndex becomes stale as settings are added | accept | Static list maintained manually alongside new settings additions; no runtime staleness possible since the list is compiled into the APK; stale entries navigate to the correct detail page even if the exact setting was renamed |
| T-06-07 | Injection | User search query processed by contains() — no injection surface | accept | Query is used only for in-memory `String.contains(ignoreCase = true)` filtering against a compile-time constant list; no SQL, no network, no eval; there is no injection vector |
| T-06-08 | Information Disclosure | HighlightedText renders user-typed query in the UI | accept | The query is only ever shown back to the user who typed it; no persistence, no logging, no server call; no privacy risk |
